Exploring cash use as a transit fare payment in Seattle

Our group is interested in cash use as fare payment method in the Greater Seattle area.

Using the Seattle Sound Transit Data of October-November 2015, we will try to answer the following questions: 1) Are cash payments still have a big stake in transit fare types among riders in Seattle? 2) What type of fare is the most common given different purpose of trips? 3)…..

Loading neccessary packages, calling dataset and preparing it for analysis

First we need to call the libraries to be able to use the packages

library(ggplot2)
library(scales)

Then we need to call the data we want to analyze. In our case we call the csv file

linkdata <- read.csv("https://raw.githubusercontent.com/karllovepolicy/Data_Visual_Final_Tutorial/master/FinalData.csv", stringsAsFactors=FALSE)

Finalizing the data frame for ggplot

We will need to get rid of missing values so that they do not distort our analysis. Then we will need to separate data we are interested to analyze (subsetting the data), and calculate the frequences

linkdata_clean <- linkdata %>%
  filter(faretype != "#NULL!")

absol_ft=table(linkdata_clean$faretype, exclude = 'nothing')
names(absol_ft)[2]='Other'
ft_freq=prop.table(absol_ft)*100
ft_freq_df=as.data.frame(ft_freq) 
names(ft_freq_df)=c("faretype","pct")
ft_freq_df_ord=ft_freq_df[order(-ft_freq_df$pct), ]

Building a bar plot using ggplot

In this section, we will explore the realationship among the probability of using cash for this trip, whether people have vehicles, the purpose of this trip, and frequency of using public transportation.

Preparing a dataset for regression

First, we need to make our dataset ready for regression. At the following code we select dependent variables and explanatory variables, then set the right format for each of the variables:

#change names of explanatory variables
library(dplyr)
mydata1 <- rename(linkdata,'number_of_trips'='oneway_all_clean', 'vehicle'='hhveh_clean')

# transform explanatory varibles from factor to numeric, add "-1" in command because Factors in R are represented internally as integers and the first factor is 1.
mydata1$vehicle <- as.numeric(mydata1$vehicle)-1
mydata1$number_of_trips <- as.numeric(mydata1$number_of_trips)-1

# subset, select needed variables:
mydata2 <- mydata1 %>% mutate(cashlove = ifelse(faretype=="Cash", 1, 0), vehicle = ifelse(vehicle>0, 1, 0)) %>% select(cashlove, purpose, number_of_trips, vehicle) %>% filter(purpose != "#NULL!")
# set the correct type for explanatory variables
mydata2$cashlove <- factor(mydata2$cashlove)
mydata2$vehicle <- factor(mydata2$vehicle)
mydata2$purpose <- factor(mydata2$purpose) #reset the level of variable "purpose"
mydata2$number_of_trips <- as.numeric(mydata2$number_of_trips)

Using the logit model

Here we use logit model since our dependent variable is a binary variable. Our reference group is people whose purpose of this trip is home-based non-work trip and don’t have vehicles at home.

library(dotwhisker)
library(broom)
#just use purpose as explantary variable
mylogit1 <- glm(cashlove ~ purpose, data = mydata2, family = "binomial")
model1 = tidy(mylogit1) %>%   # we save the result as a tidy object and...
    mutate(model = "Model 1")
#use purpose and number of trips
mylogit2 <- glm(cashlove ~ purpose + number_of_trips, data = mydata2, family = "binomial")
model2 = tidy(mylogit2) %>%   # we save the result as a tidy object and...
    mutate(model = "Model 2")
#use all three explanatory varibales
mylogit3 <- glm(cashlove ~ purpose + number_of_trips + vehicle, data = mydata2, family = "binomial")
model3 = tidy(mylogit3) %>%   # we save the result as a tidy object and...
    mutate(model = "Model 3")

Comparing three models

allModels=rbind(model1, model2, model3)

dwplot(allModels, vline = geom_vline(xintercept = 0, colour = "grey60", linetype = 2)) + 
  xlab("Coefficient Estimate") +

  ggtitle("Predicting probability of using cash") +

From the plot above, we can see that compared with reference group, people whose purpose is home based work trip have less probability using cash, while people whose purpose is non-home based trip is more likely to use cash. The more trips people have in each month, the less likelihood people would use cash. However, the difference on cash usage between people who have vehicles in home and people who don’t is non-significant.
